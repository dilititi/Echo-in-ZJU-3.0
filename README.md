# 交互式校园地图

这是一个基于 Leaflet.js 的交互式校园地图应用。

## 使用说明

1. 将你的学校地图图片重命名为 `map.jpg` 并放在项目根目录下
2. 在 `icons` 文件夹中放入以下图标文件：
   - classroom.png（教室图标）
   - library.png（图书馆图标）
   - cafe.png（餐厅图标）
   - sports.png（体育设施图标）

## 功能特点

- 在地图上自由添加各种设施图标
- 拖拽移动已添加的图标
- 右键点击删除不需要的图标
- 简单直观的图标选择面板

## 使用方法

1. 从右侧面板选择要添加的图标类型
2. 点击地图上的任意位置添加图标
3. 拖拽图标可以调整位置
4. 右键点击图标可以删除

## 运行方法

使用任意 Web 服务器在本地运行，例如 Python 的 http.server：

```bash
python -m http.server 8000
```

然后在浏览器中访问 http://localhost:8000
