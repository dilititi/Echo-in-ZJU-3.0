📖 使用帮助

  • 🗺️ 点击地标：播放校园故事音频
  • 📌 放置标记：选择类型后按 N 键
  • 🗑️ 删除标记：选中后按 Backspace
  • 🎤 录制声音：在 Field Notes 卡片按 ● 录音并命名
  • 📁 上传谜题：⋯ 菜单 → 创作 → 上传音频谜题（≤20MB）
  • ✨ 创建事件：⋯ 菜单 → 创作 → 自定义事件
  • 🎧 声音探索：⋯ 菜单 → 玩法 → 听音辨物
  • 🗺️ 声音游记：⋯ 菜单 → 玩法 → 一键全播
  • 💬 社区：⋯ 菜单 → 社交 → 留言墙
  • 🏙️ 3D 校园：⋯ 菜单 → 视图 → 3D 校园

在浏览器控制台运行 checkConfig() 可查看完整配置检查报告。

4. 答对即解锁地标，探索整张声音地图
