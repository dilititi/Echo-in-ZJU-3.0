  • 🗑️ 删除标记：选中后按 Backspace
  • 🎤 录制声音：在右侧面板录音并命名
  • 📁 上传谜题：支持 .wav / .mp3 / .ogg / .m4a（≤20MB）

  • 🎧 声音探索：工具栏开启，听音辨物解锁地标
  • 🗺️ 声音游记：查看你留下的所有声音时间线
  • 💬 社区：留言声音墙 + 情绪排行榜
  • 🌙 日/夜模式：工具栏一键切换，偏好自动保存

声音探索模式

1. 地图地标全部隐藏，等待你来解锁

2. 点击地图蓝色热点区域，播放环境音频

3. 根据声音特征，猜测并输入建筑名称

4. 答对即解锁地标，探索整张声音地图
